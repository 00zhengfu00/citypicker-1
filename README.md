## citypicker

### 写在前面的话
在实际的项目中需要使用到省市区三级联动的功能，在网上找来找去，都没有找到一个合适的库，本来使用了[“Android-PickerView”](https://github.com/saiwu-bigkoo/Android-PickerView)的，后来写到最后发现里面有一些问题，所以放弃了，不过还是得感谢以下，参考了里面的源码。最后在网上查找了，使用一个半成品，最后加工成成品。
